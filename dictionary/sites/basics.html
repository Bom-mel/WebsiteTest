Bommels HTML-Dictionary

Basics:

Du benutzt HTML für die Grundstruktur einer Website.

So wie diese hier!


Damit der Server weiß das du HTML nutzt braucht der Code ein "Grundgerüst"

Das sieht wie folgt aus:

website.html:
<!DOCTYPE html>
<html>
<head>
   <title>Website Titel</title>
   <meta ...>
</head>
<body>
</body>
</html>

Was ist <!DOCTYPE html> ?

Es sagt dem Browser: das ist eine HTML5-Seite um sicherzustellen das er sie im Standard-Rendering-Modus darstellt.

Was ist <head> ?

Es enthält die Metadaten die nicht direkt auf der Website sichtbar sind, aber für den browser und die Suchmaschine wichtig sind.

Was ist <body> ?

Es enthält alle sichtbaren und interargierbaren Inhalte. Also z.B. Texte, Bilder, Links, ...
